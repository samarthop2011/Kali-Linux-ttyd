# Kali-Linux-ttyd
# Kali Linux Web Terminal for Railway

This project deploys a Kali Linux container with ttyd web terminal on Railway.

## Deploy

1. Push this repo to GitHub
2. Open Railway
3. Create New Project
4. Deploy from GitHub Repo
5. Railway will automatically build and expose the web terminal
